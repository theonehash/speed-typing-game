# Speed Typing Game

A simple speed typing game built using HTML, CSS, and JavaScript. The game challenges players to type a given sentence as quickly and accurately as possible while displaying their speed as a percentage of the world record.

## Features
- Random sentence selection
- Real-time error detection (prevents further typing if incorrect)
- Timer to track typing speed
- Displays typing speed as a percentage of the world record (216 WPM)
- Restart button to try again

## How to Play
1. Start typing the displayed sentence in the input field.
2. If you make a mistake, the input field turns red, and you must correct it before proceeding.
3. Once you complete the sentence, your time and typing speed percentage will be displayed.
4. Click "Restart" to try again with a new sentence.

## Installation
1. Clone this repository:
   ```sh
   git clone https://github.com/theonehash/speed-typing-game.git
   ```
2. Navigate to the project directory:
   ```sh
   cd speed-typing-game
   ```
3. Open `index.html` in a browser to play the game.
